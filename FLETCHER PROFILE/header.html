Static

Use of HTML5 and CSS for Front end.

It is the most populous type of simple website that displays the same content to all users, without any dynamic or interactive elements.

In other words, the content remains fixed and unchanging, and users can only consume the information presented to them. These websites are typically coded using only HTML, CSS.

Dynamic

Combines both front-end and back-end.

A dynamic website is a type of website that delivers content to users in a flexible and interactive manner.

Unlike static websites that present fixed content to all visitors, dynamic websites generate content on the fly,

This is achieved through the use of server-side scripting languages and databases, allowing websites to respond dynamically to user inputs and deliver personalized content.
